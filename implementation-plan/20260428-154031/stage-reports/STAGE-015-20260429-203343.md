## Changes made
- Added `src/http/cors.py` as the shared CORS contract source for allowed methods, allowed request headers, exposed response headers, origin parsing, and preflight helpers.
- Updated response building to emit only a single browser-valid ACAO value, append `Vary: Origin` for reflected origins, include `SMUGGLE`, expose `X-Request-Id`, `X-Smuggle-URL`, `X-File-Modified`, and `ETag`, and allow implemented request headers including `Authorization` and `If-None-Match`.
- Updated request processing to resolve CORS per HTTP request `Origin` before response serialization, so multi-origin configs echo only the matching request origin and missing/unlisted origins emit no ACAO.
- Updated WebSocket origin checks to use the same parsed origin allowlist and reject mixed wildcard-plus-explicit configs.
- Updated OPTIONS handling so unknown methods are advertised only when advanced upload is enabled, while requested CORS headers are filtered to implemented header names.
- Added regression tests for multi-origin match/miss, missing origin, wildcard, mixed wildcard rejection, preflight advanced-upload gating, auth/conditional request headers, and exposed headers.
